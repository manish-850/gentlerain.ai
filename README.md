# Gentlerain.ai UI Clone

A front-end UI clone inspired by the Awwwards‑winning website
**gentlerain.ai**.\
This project focuses purely on replicating the visual layout,
animations, and scrolling experience for learning purposes.

## Overview

This clone aims to recreate the design and flow of the Gentlerain
website, including: - Smooth scrolling interactions.
- Canvas-based hero section
- Lottie animation integration
- Section‑based storytelling layout
- Clean typography and structured content blocks

## Preview

![preview](./img/gentlerain_preview.gif)

## Features

-   Hero section with canvas animation
-   Integrated Lottie animation
-   Multi‑section layout with alternating text + image blocks
-   Modern navigation bar with responsive layout foundation
-   Smooth scrolling using **Lenis**
-   Clean and minimal UI resembling the original site

## Tech Stack

-   HTML5
-   CSS3
-   JavaScript
-   Lenis (smooth scroll)
-   Lottie Animation (iframe-based)
